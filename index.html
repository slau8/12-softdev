Getting Started

There are two ways to use Bootstrap for your project:

  • Download from getbootstrap.com
  • Include Bootstrap from a Content Delivery Network

The second option, being faster and more convenient, is most optimal. Here is a base template: 

       <!DOCTYPE html>
        <html lang="en">
          <head>
            <title></title>
            <meta charset="utf-8">
            <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.7/css/bootstrap.min.css">
            <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.2.0/jquery.min.js"></script>
            <script src="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.7/js/bootstrap.min.js"></script>
          </head>
          <body>
          </body>
        </html>

Text  typography, formatting

Include these tags or class attributes to format text:


Class/Element Description
.list-inline Places all list items on a single line
.text-left Aligns text left
.text-center Aligns text center
.text-right Aligns text right
.text-justify Aligns text justified
<code> Indicates code
<mark> Highlights text
<small> Creates lighter, secondary text in any heading

Layout  containers, grids

Containers

The .container class provides a responsive fixed width container:

        <div class="container">
          ...
        </div>

The .container-fluid class provides a full width container:

        <div class="container-fluid">
          ...
        </div>
These classes are used within <div> tags.

Grids

Bootstrap utilizes a 12-column system to organize HTML. Within <div class="row">, there can be as many as 12 columns whose widths sum to 12. The class for each of these columns is col-lg-* where * represents the width of the column. lg references the size for laptop/desktop screens at least 1200px wide. 

        <div class="row">
          <div class="col-lg-6">.col-lg-6</div>
          <div class="col-lg-6">.col-lg-6</div>
        </div>
        <div class="row">
          <div class="col-lg-4">.col-lg-4</div>
          <div class="col-lg-8">.col-lg-8</div>
        </div>
.col-lg-6
.col-lg-6

.col-lg-4
.col-lg-8

Note: Any configuration of columns, whether there be one, six or twelve columns, can be used as long as the total width equals 12.

Display  tables, images, page-headers, buttons, navbars

Include these class attributes for different ways to display your site:


Class Description
.btn Creates standard Bootstrap button
.btn-block Creates a full-width button
.img-rounded Rounds the corner of an image
.img-circle Shapes image into a circle
.img-thumbnail Displays image as a thumbnail
.page-header Adds a horizontal line and padding under a heading
.table Creates standard Bootstrap table
.table-striped Creates table with zebra-striped rows
.table-hover Highlights the row that the mouse is over
